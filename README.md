# TCP Port Scanner

A simple TCP port scanner written in C#.

## Features

- Scan a single IP address or hostname
- Check whether a TCP port is open or closed
- Uses TcpClient to establish TCP connections

## Technologies

- C#
- .NET
- TCP Networking

## Example

Enter IP address:
google.com

Enter port:
80

Output:

Port 80 is OPEN.

## Future Improvements

- Scan a range of ports
- Add connection timeout
- Display common service names
- Scan ports asynchronously
- Export results to a file
